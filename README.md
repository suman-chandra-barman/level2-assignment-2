## Install Dependencies:
npm install

## Available Scripts

### Start the Application (Production):
npm start

### Start the Application (Development):
npm run start:dev

### Build the Project:
npm run build

## Linting

### Run ESLint:
npm run lint

### Fix ESLint Issues:
npm run lint:fix

## Prettier

### Format Code with Prettier:
npm run prettier

### Fix Prettier Issues:
npm run prettier:fix
